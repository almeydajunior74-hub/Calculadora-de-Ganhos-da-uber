Aplicação web simples desenvolvida para auxiliar no controle e cálculo de ganhos diários trabalhando com Uber. O objetivo do projeto é facilitar a visualização de faturamento e apoiar na organização financeira como renda complementar.

Sobre o Projeto

Este aplicativo foi criado para uso próprio, permitindo registrar e acompanhar os ganhos diários de forma prática. A solução ajuda a entender melhor o desempenho financeiro ao longo dos dias trabalhados.

Funcionalidades
Registro de ganhos diários
Cálculo automático de faturamento
Visualização simples e objetiva dos resultados
Interface leve e fácil de usar
Tecnologias Utilizadas
HTML
CSS
JavaScript
Objetivo

O projeto foi desenvolvido com foco em praticar lógica de programação e desenvolvimento web, além de resolver uma necessidade real de controle financeiro pessoal.
